# Imposter Hunt

An iOS game built with SwiftUI where players must find the imposter among them.

## Requirements

- iOS 18.5+
- Xcode 16.4+
- Swift 5.0

## Building

```bash
# Open in Xcode
open "Find The Imposter.xcodeproj"

# Or build from command line
xcodebuild -project "Find The Imposter.xcodeproj" -scheme "Find The Imposter" -configuration Debug build
```

## Project Structure

- **Find The Imposter/** - Main source files
- **Assets.xcassets/** - App icons and images
- Built with SwiftUI using declarative UI patterns

## Development

The app uses SwiftUI Previews for rapid iteration. Use the `#Preview` macro to preview UI components in Xcode.
